# Pipeline Scheduler MVP

Pipeline Scheduler MVP schedules retry tasks from retry plans generated by the Pipeline Retry Controller.

It calculates execution timestamps, builds execution queues, exports JSON schedules, and provides a CLI for inspecting scheduled tasks.

Part of the RADAR Services utility chain.

## CLI

Schedule retry tasks

```bash
pipeline-scheduler schedule data/sample_retry_plan.json
```

Verbose

```bash
pipeline-scheduler schedule \
    data/sample_retry_plan.json \
    --verbose
```

Generate JSON schedule

```bash
pipeline-scheduler schedule \
    data/sample_retry_plan.json \
    --json
```

Custom output

```bash
pipeline-scheduler schedule \
    data/sample_retry_plan.json \
    --json \
    --output outputs/schedule.json
```
